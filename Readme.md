# WatcherBot

Бот для отслеживания кнопки принять в Dota 2 и Counter-Strike. WatcherBot ищет заданный шаблон на экране и при обнаружении отправляет уведомление в Telegram. Также позволяет принимать игру через бота.

---

## Настройка приложения

## 1. Клонирование репозитория и установка окружения

```bash
git clone https://github.com/Mishcsas/WatcherBot
cd WatcherBot
python -m venv venv
```

Активируйте виртуальное окружение:
- Windows: `venv\Scripts\activate`
- Mac/Linux: `source venv/bin/activate`

## 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 3. Создание файла `needs.json`

Создайте в корне проекта файл `needs.json` со следующей структурой:

```json
{
  "PROXY": "http://ваш_прокси:порт",
  "TOKEN": "ваш_токен_бота",
  "USER_ID": 0,
  "TRIGER" : 0.5
}
```
`TRIGER` - число на которе будет тригерится принмает значение от 0 до 1 чем выше значение тем больше нужно совпадение для тригера.
### Получение токена
- Найдите в Telegram бота **BotFather**
- Отправьте команду `/newbot`
- Следуйте инструкциям и получите токен
- Вставьте токен в поле `TOKEN` (кавычки оставьте)

### Настройка прокси
- В поле `PROXY` укажите URL вашего прокси-сервера для Telegram

### Поле USER_ID
- Необязательное поле
- Бот создаст ID автоматически, или можете указать `0`

## 4. Добавление шаблонов

В папку `templates` добавьте все необходимые шаблоны (изображения). Рекомендуется использовать разные имена файлов.

## 5. Запуск приложения

```bash
python main.py
```

## Примечания
- Убедитесь, что все зависимости установлены корректно
- Проверьте правильность формата JSON в файле `needs.json`
- Убедитесь, что папка `templates` существует и содержит нужные файлы
